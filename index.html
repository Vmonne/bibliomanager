Zone de réinitialisation

⚠️ Ces actions sont irréversibles. Exportez vos données avant toute réinitialisation.

📱 Installer sur téléphone Android
  1. Ouvrez l'application dans Google Chrome
  2. Appuyez sur le menu ⋮ (trois points en haut à droite)
  3. Appuyez sur "Ajouter à l'écran d'accueil"
  4. Confirmez → l'icône BiblioManager apparaît sur votre écran ✅
🍎 Installer sur iPhone
  1. Ouvrez l'application dans Safari
  2. Appuyez sur l'icône Partager (carré avec flèche)
  3. Appuyez sur "Sur l'écran d'accueil"
  4. Appuyez sur "Ajouter" → l'icône apparaît ✅
💻 Installer sur ordinateur
  1. Ouvrez dans Google Chrome ou Microsoft Edge
  2. Cliquez sur l'icône ⊕ dans la barre d'adresse
  3. Cliquez sur "Installer" → l'app s'ouvre comme native ✅
💾 Transférer mes données
  1. Sur l'appareil source → cliquez "Sauvegarder" en haut
  2. Un fichier bibliomanager_backup.json est téléchargé
  3. Envoyez ce fichier sur le nouvel appareil (WhatsApp, email…)
  4. Sur le nouvel appareil → cliquez "Synchroniser" et sélectionnez le fichier ✅
